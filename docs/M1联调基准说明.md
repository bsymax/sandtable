| 前端 | `intelligence.html` | 主工程 **`web/intel.html`**：品牌筛选、`?brand=` 整页过滤、预警/新闻/周报 **✏️ 编辑** |
| 迁移脚本 | — | `database/migrate_intel_unify.sql`（已有库升级用） |

**开开 M2 开发必须以 `synced-from-main/server/routers/intel.py` 为准**，0611 的 `IntelWeeklyReport` 模型勿再扩展。

### 3.6 API 根地址（**2026-06-13 新增 · 全员须知**）

| 环境 | 行为 |
|------|------|
| **本机** 5510 静态 + 8000 API | `web/js/api-base.js` → `M1_API_BASE = http://127.0.0.1:8000` |
| **外网** Nginx 同源 | `M1_API_BASE = ''`，请求走 `/api/*` |
| 涉及文件 | `index.html`、`profile.html`、`visit.html`、`intel.html` 均引用 `api-base.js` |

> 若页面报 `Unexpected token '<', "<!DOCTYPE"...`，即 API 打到了静态服务器而非 8000/反代——检查是否漏引 `api-base.js` 或硬刷新缓存。
