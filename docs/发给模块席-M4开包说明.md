## 2. M4 和 M3 有什么不同

| 项 | M3（已收官） | M4 |
|----|-------------|-----|
| 登录/角色 | 有登录 + 品牌过滤 | **`readonly` 禁写** · 四角色全量回归 |
| LLM | 五路开关 + 降级 | **审计日志 + 配额** · Prompt 调优 |
| 数仓 | CSV 管道 + 样例数据 | **内网 BI 首接真 GMV**（≥1 品牌） |
| 档案 | Tab1/Tab2 LLM | **数据截至 Wxx** · 佳璇牵头 BI 对照 |
| 拜访 | 纪要 LLM | **培翛牵头联调** · 「数据已更新至 Wxx」 |
| 工程 | 外网主验收 | **泰山预发**（M4-D）· 驾驶舱 → **M6+** |
| 排期 | 单窗口 ~8 日 | 同上 · 按 **M4-A～D** 里程碑解锁 |
