## 3. 字段口径（Excel 列名 → CSV）

### 必填

| 界面名称 | CSV 列名 | Excel 建议列名 | 单位 |
|----------|----------|----------------|------|
| **成交** | `gmv` | 成交 / 月成交（**JD 渠道**） | **万元** |
| **成交同比** | `gmv_yoy` | 成交同比（**JD 渠道**） | % |
| 销量 | `sales_volume` | 销量（**JD 渠道**） | 件 |
| 销量同比 | `sales_volume_yoy` | 销量同比（**JD 渠道**） | % |
| 月份 | `period_value` | 月份 | `2026-05` |
| — | `period_type` | 固定 | `monthly` |
| — | `name_key` | 品牌键 | 见 §2 |

### 建议有

| 界面/用途 | CSV 列名 | Excel 建议列名 |
|-----------|----------|----------------|
| 京东市占 | `jd_share` | 京东市占% |
| 京东市占变化 | `jd_share_wow` | 京东市占变化 |
| 天猫/抖音/淘宝市占 | `tmall_share` / `douyin_share` / `taobao_share` | 天猫市占% / 抖音市占% / 淘宝市占% |
| 天猫/抖音/淘宝渠道增速 | `channel_growth_tmall` / `channel_growth_douyin` / `channel_growth_taobao` | 渠道增速 |
| 动销商品数(SPU) | `sku_count` | 动销商品数(SPU) |

### 不必维护（或 seed 示范）

`gmv_wow`、`sales_volume_wow`、`channel_growth_jd`、`pdd_share`（已改为 `taobao_share`）

**底表暂无、本地 seed 自动生成【示范数据】**：`p0_gap_count`（P0 缺口 SKU 明细仍为前端示范）

**二级类目（真实 · Excel「类目数据源」）**：`category_distribution`、`category_share`  
导入：`python3 data/dw/transform_category_source.py` → `brand_category_monthly.csv` → `seed.py`

| JSON 字段 | 含义 |
|-----------|------|
| `category_distribution[].share` | 品牌下类目占比 = 二级品类四渠道销售 / 品牌四渠道销售合计 |
| `category_share[].jd_brand_share` | JD品类占比 = 二级品类 JD 销售 / 品牌 JD 销售合计 |
| `category_share[].jd/tm/tb/dy` | 该二级品类内，各渠道销售占四渠道合计 |

样例：`data/dw/brand_category_monthly.csv`

### 源表透视（品牌档案数据源.xlsx）

长表每行 = **品牌 × 月份 × 渠道**。渠道值映射：

| 源表「渠道」 | 数仓字段 |
|--------------|----------|
| `JD` | `jd_share`（市占率×100）、`jd_share_wow`（市占率同环比×100） |
| `A-TM(大贸+国内现货)` | `tmall_share`、`channel_growth_tmall`（销售额同环比×100） |
| `A-TB(非国际)` | `taobao_share`、`channel_growth_taobao` |
| `N-DY(非国际)` | `douyin_share`、`channel_growth_douyin` |

品牌级 KPI（**均取 JD 渠道**；其他渠道仅用于 § 建议有 中的市占/增速）：

| 源表列 | 数仓字段 | 规则 |
|--------|----------|------|
| 销售额 | `gmv` | JD 销售额 ÷ 10000（**万元**） |
| 销售额同环比 | `gmv_yoy` | JD 销售额同环比 × 100 |
| 销量 | `sales_volume` | JD 销量（件，四舍五入取整） |
| 销量同环比 | `sales_volume_yoy` | JD 销量同环比 × 100 |
| 商品数 | `sku_count` | JD 商品数（动销商品数 SPU） |
| 时间 | `period_value` | `YYYY-MM` |

转换脚本：`python3 data/dw/transform_brand_source.py`
